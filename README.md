# AMQDo

A Rust ActiveMQ client crate.

[![Build Status](https://travis-ci.org/russmack/amqdo.svg?branch=master)](https://travis-ci.org/russmack/amqdo)

---
#### Status: PoC.
----

## Build
```
git clone https://github.com/russmack/amqdo.git
cargo build --release
```

## Usage
```
cargo run --release
```
